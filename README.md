# Customer Feedback Dashboard

A Streamlit dashboard that analyzes customer feedback using TextBlob Sentiment Analysis.

## 🚀 Live Demo

https://customerfeedbackdashboard-7iwappeqgbvtr6g9dkb8fbd.streamlit.app

## Features

- Add customer feedback
- View feedback records
- Sentiment Analysis (Positive, Negative, Neutral)
- Interactive Plotly charts

## Installation

```bash
pip install -r requirements.txt
```

## Run Locally

```bash
streamlit run streamlit_app.py
```
