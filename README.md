# 📊 Customer Feedback Dashboard

An AI-powered Customer Feedback Dashboard built with **Streamlit**, **Python**, **Plotly**, and **TextBlob** to analyze customer reviews using sentiment analysis.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AaravSingh04/Customer-Feedback-Dashboard.git
```

Move into the project

```bash
cd Customer-Feedback-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run streamlit_app.py
```
